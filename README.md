# Yucatan PropTech AI

**AI-powered real estate investment platform for the Yucatan Peninsula**

Yucatan PropTech AI democratizes real estate investment in Mexico's Yucatan Peninsula by leveraging Claude AI to guide users through the complex fideicomiso (bank trust) process required for foreign property ownership.

## Problem

Foreign investors face significant barriers when investing in Mexican real estate:
- Complex fideicomiso (bank trust) requirements in restricted zones
- Language barriers and lack of transparent market data
- Fragmented property listings across multiple sources
- No unified platform for AI-assisted investment decisions

## Solution

An open-source AI platform that:
- **Guides fideicomiso process** - Claude AI provides step-by-step guidance
- **Analyzes properties** - AI-powered property valuation and investment scoring
- **Provides market insights** - Real-time analysis of Yucatan real estate trends
- **Multilingual support** - Seamless English/Spanish interaction

## Quick Start

Prerequisites: Python 3.10+, Node.js 18+, Anthropic API key

## Claude AI Integration

This project uses the Anthropic Claude API for:
1. Conversational Fideicomiso Guide
2. Property Analysis
3. Document Review
4. Market Intelligence
